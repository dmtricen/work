Reload Passport
- USB in port
- Restart
- Hit f7 to enter bootmenu
- password : passasc 
- Select passport image


Update order
1. Passport Software
2. Maintenance Packs
3. CPI cert
4. Sectigo cert

Note: call TAC and understand the upgrade path for the processor you are using
(Or, lookup network addendum for processor under “quick fix”)

Passport upgrade document is SB3211

To update/install maintenance packs, copy them into the folder D:/Gilbarco/SWUpgrade/Download/

After a few seconds it will disappear and begin installation.
